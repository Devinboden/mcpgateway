# Boom AgentCore Assets

A point-in-time mirror of the **Boom MCP** as deployed to **Amazon Bedrock AgentCore Runtime**, kept
here in the mcpgateway repo for reference. This is a *snapshot*, not the source of truth — the live
source of truth is the Boom MCP repo.

## Source
- **Repo:** https://github.com/Noland-LAL/boom-mcp.git
- **Snapshot of:** `origin/main` @ commit **`cb2f84d`** ("Trim Boom tool surface: 11 → 5 tools").
- The application source here (`src/`, `app/`, `widget/`, `vendor/`, `data/`, `package*.json`) is an
  exact extract of that commit.

## Notes / caveats
- **`.bedrock_agentcore.yaml` contains machine-specific absolute paths** (`entrypoint`, `source_path`)
  and account ARNs. These are regenerated by `agentcore configure` on the deploying machine; treat the
  paths as illustrative.
- **The live runtime image may lag this snapshot.** The deployed image was built from an earlier local
  commit; this asset set reflects the *current* Boom repo (`origin/main` cb2f84d). Rebuild via CodeBuild
  to bring the runtime up to date.
- The Boom repo's local `Dockerfile.agentcore` carried a base-image tweak
  (`FROM public.ecr.aws/docker/library/node:22-slim`, to avoid Docker Hub rate limits) not yet merged
  to `origin/main`; the version here is `origin/main`'s. The actual CodeBuild build uses
  `.bedrock_agentcore/boom_mcp/Dockerfile`.
- To refresh this folder: re-run `git archive origin/main` from the Boom repo and re-overlay the
  `.bedrock_agentcore*` files.
